# Portfolio: Webauftritt für gesunde Rezeptideen

## Projektübersicht

Im Rahmen dieses Projektes wird eine responsive Webseite für **gesunde und schnell zubereitete Rezeptideen** entwickelt. 
Der Webauftritt besteht aus einer Startseite mit vier verlinkten Rezeptseiten sowie einer zusätzlichen Informationsseite zu ausgewählten Zutaten. 
Ziel ist es, nicht nur gesunde Rezepte vorzustellen, sondern auch **Hintergrundinformationen zu den verwendeten Zutaten**, deren Nährwerte und mögliche positive Auswirkungen auf eine aufgewogene Ernährung bereitzustellen. 

Der Umfang und die Komplexität können im Verlauf der Entwicklung erweitert werden. 

## Aufbau des Webauftritts

### Startseite

Die Startseite dient als zentraler Einstiegspunkt in den Webauftritt. Sie besteht aus: 
- einer Hauptnavigation 
- einem Hero-Bereich mit Bild und Slogan 
- Rezeptkarten mit Kurzbeschreibung und Verlinkungen zu den Rezeptseiten 
- einem Footer mit Kontaktinformationen 

### Rezeptseiten

Jede Rezeptseite folgt einer einheitlichen Struktur und enthält:
- Hauptnavigation 
- Rezeptbild 
- Rezeptbeschreibung 
- Angaben zur Portionsgröße
- Zutatenliste 
- Zubereitungsschritte
- Rezeptnavigation zur Startseite sowie zu vorherigen bzw. nächsten Rezepten 
- Footer mit Kontaktinformationen 

### Informationsseite 

Die Informationsseite stellt ergänzende Informationen zu den verwendeten Zuaten bereit. Die Inhalte werden kategorisch gegliedert nach: 
- Kohlenhydraten 
- Proteinquellen 
- Gemüse 
- Früchten 
- Kräutern 

Ziel ist es, die gesundheitlichen Eigenschaften und Nährwerte der verwendetetn Zutaten verständlich darzustellen. 

## Technische Umsetzung 

### HTML

Für die Strukturierung der Inhalte wird semantisches HTML verwendet. Dabei kommen unter anderen folgende Elemente zum Einsatz: 
```html
- <header>
- <nav>
- <main>
- <section>
- <article>
- <footer>
```
Die Verwendung semantischer Elemente verbessert die Strukturierung der Inhalte und unterstützt die Barrierefreiheit des Webauftritts. 

### CSS

CSS wird für die Gestaltung des Webauftritts eingesetzt. Dazu gehören insbesondere:
- Layoutgestaltung
- Farbkonzept 
- Typografie 
- Abstände und Positionierung 
- Umsetzung des responsiven Designs mittels Media Queries 

### JavaScript (geplante Erweiterung)

Optional ist die Entwicklung eines Portionsrechners geplant. Nutzerinnen und Nutzer sollen die Anzahl der Personen über Schaltflächen anpassen können, wodurch die Zutatenmengen automatisch neu berechnet werden. 

## Geplante Erweiterungen 
- Dynamischer Portionsrechner mit JavaScript 
- Zusätzliche Informationen zu einzelnen Zutaten 
- Erweiterung um weitere gesunde Rezeptideen 
- Verbesserung der Benutzerfreundlichkeit und Barrierefreiheit
